Sprint M2 · Mobile UI

Behbehani Motors — Customer App

Production-grade mobile mockups (375 px iPhone-13 baseline). Royal Blue #1E3A8A + Plus Jakarta Sans (Latin) / Tajawal (Arabic). KWD shown with 3 decimals. VIN last-6 masked. Touch targets ≥ 44 px.

RTL audit

Toggle <html dir="rtl"> in each file to verify the flip. Every layout must read right-to-left in Arabic without horizontal scroll, clipped icons, or mis-aligned chevrons. We use logical Tailwind props (ps-*, me-*, text-start) throughout.

Locked design rules honored

  • Customer brand: Royal Blue #1E3A8A + white + slate neutrals only. No amber/yellow/gold/emerald/green (red-50/red-700 for warnings).
  • Fonts: Plus Jakarta Sans (Latin) and Tajawal (Arabic).
  • Money: KWD 5.250 (3 decimals, never fils, never integer).
  • VIN: last 6 only on customer surfaces (e.g. ··· ··· A12345).
  • Tab bar: Home / Buy / Sell / Services / Account + More (5 visible + More sheet for Finance / Maintenance / Favorites).
  • Sign-in: Email+password active; OTP/Google/Apple disabled with "Coming soon".
  • Touch targets ≥ 44 px; safe-area top + bottom respected.
